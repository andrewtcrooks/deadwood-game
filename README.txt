# Deadwood

This is a Java application for the game Deadwood.

## Prerequisites

You need to have Java Development Kit (JDK) 17 installed on your machine. You can download it from the official [Oracle website](https://www.oracle.com/java/technologies/javase-jdk17-downloads.html).

## Compilation

1. Open a terminal.
2. Navigate to the `src` directory containing the source files. 
3. Compile the Java files into bytecode (.class files) using the `javac` command:

```bash
cd src
javac *.java
